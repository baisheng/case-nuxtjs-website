# wepy2-homepage

> 为 WePY2 做的官方网站

##
[👉 在线示例](https://blissful-swanson-78f1fa.netlify.com/)

## 基于 nuxtjs 构建

- pixijs 实现案例动态案例展示
- nuxtjs generate 生成静态页面

## 构建步骤

``` bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn run dev

# build for production and launch server
$ yarn run build
$ yarn start

# generate static project
$ yarn run generate
```
